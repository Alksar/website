Данный сайт является студенческим проектом, отражающим навыки использования таких технологий, как HTML (в том числе HTML5), CSS (в том числе CSS3) и JavaScript.

Сайт разработан с помощью современных методов, масштабируем, переносим, легко управляем, что достигнуто при помощи грамотного использования следующих инструментов и их функций:

  • HTML - разметка страницы, регулирующая расположение различных блоков информации на сайте.
  • CSS - визуальное гармоничное оформление, радующее ваши глаза и подчеркивающее чувства стиля.
  • JavaScript - функционал, обеспечивающий внутренние размышления сайта чуть более, чем на 100%.
